## Features
- Login Page
- Register Page
- Dockerize a application 
- Supports multiple languages
- Supports Send Email in Queue

## Installation
  - Run `bower install`
  - Run `npm install`


## Run
  - Run `npm start`
  - Run `grunt test` to make sure the application work perfectly
